[![Chrome Install](https://img.shields.io/badge/Chrome-Install-green?logo=googlechrome)](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag)

Once the extensions are installed individual scripts/styles can be installed via clicking the associated install button below.
